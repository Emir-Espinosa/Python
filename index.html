Qué aprenderás

En este curso de Python aprenderás a crear, proyectos, paso a paso, de manera profesional y sencilla. También, aprenderás a dominar las diferentes herramientas del lenguaje para llevar un flujo de trabajo más ágil y eficiente.

Al finalizar este curso de Python podrás:

  • Objetivo 1. Comprender y utilizar los fundamentos de Python.
  • Objetivo 2. Aplicar programación orientada a objetos.
  • Objetivo 3. Utilizar bibliotecas y paquetes externos.
  • Objetivo 4. Prepararte para oportunidades profesionales en Python..
  • Objetivo 5. Desarrollar y presentar proyectos prácticos.
  • Objetivo 6. Optimizar y seguir buenas prácticas..
